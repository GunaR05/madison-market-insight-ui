# Madison Market Insight Engine (Assignment 5 UI)

Streamlit interface that wraps the Assignment 4 n8n workflow output for non-technical users.

## What it does
- Upload the A4 output JSON (madison_job_dataai.json)
- Displays executive-ready insights in formatted sections
- Shows run metadata and simple charts

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
